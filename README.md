## Requirements

* PHP 5.3.3
* PostgreSQL
* php app/check.php

## Installation

```
git clone https://github.com/lassiheikkinen/pulupalsta.git palsta.pulu.org
cd palsta.pulu.org
curl -sS https://getcomposer.org/installer | php
vim app/config/parameters.yml
php composer.phar install
```
